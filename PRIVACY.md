LingoNote Privacy Policy

Last Updated: 2026-02-26

Data Collection: LingoNote does not collect, store, or transmit any personal data or browsing history to external servers.

Data Storage: All notes and highlights created by the user are stored locally on the user's device using the browser's local storage.

Permissions: The extension only requests necessary permissions to provide its core features (highlighting and note-taking).

Contact: For any questions, please contact the developer via GitHub Issues.

隐私政策

数据收集：LingoNote 不会收集、存储或向外部服务器传输任何个人数据或浏览历史。

数据存储：用户创建的所有笔记和高亮内容均通过浏览器本地存储保存在用户设备上。

权限说明：本插件仅申请实现核心功能（划词高亮和笔记）所必需的权限。
