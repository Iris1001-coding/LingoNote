# 🚀 LingoNote - 沉浸式网页学习助手

LingoNote 是一款轻量级的浏览器插件，旨在帮助语言学习者在阅读网页时，能够随时高亮生词、添加笔记，并永久保存这些学习记录。

---

## 📥 安装指南 (开发者模式版)

由于插件目前正在申请上架 Chrome 应用商店，请按照以下步骤手动安装：

1. **下载插件包**：
   前往 [Releases 页面](https://github.com/Iris1001-coding/LingoNote/releases/tag/v1.0.2) 下载最新的 `LingoNote_v1.0.2.zip`。
   也可前往Edge商店进行搜索下载，因为目前刚刚上线没有搜索权重，推荐使用传送门； https://microsoftedge.microsoft.com/addons/detail/phbicpecddkjcfdhkadpnddhmfgbjeom 直达
   
3. **解压文件**：
   将下载好的 `.zip` 文件解压到你电脑的一个固定文件夹中（安装后请勿删除此文件夹）。

4. **进入扩展程序页面**：
   在 Chrome 浏览器地址栏输入 `chrome://extensions/` 并回车。

5. **开启开发者模式**：
   点击页面右上角的 **“开发者模式” (Developer mode)** 开关。

6. **加载插件**：
   点击左上角的 **“加载已解压的扩展程序” (Load unpacked)**，选择刚才解压出来的文件夹。

7. **完成！**
   现在你可以在浏览器工具栏看到 LingoNote 图标了，点击即可开始学习。

   🔄 如何更新版本而不丢失数据？
如果你已经安装了旧版本，请千万不要先“删除”插件！

下载最新的 LingoNote_vX.X.X.zip 并解压。

将解压后的文件覆盖到你原来的安装文件夹中。

打开 Edge 的扩展管理页面 (edge://extensions)。

找到 LingoNote，点击右下角的**“刷新 (Reload)”**图标（那个圆圈箭头）。

这样插件就会加载新代码，同时保留你之前所有的笔记数据。
